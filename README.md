

\## Call-Audit-System



\- Monitoring and analytical platform used to audit communication activities.

\- Supports multi-user login with role-based access control.

\- Processes large volumes of call data and system logs.

\- Enables administrators to monitor communication usage and billing.

\- Generates audit reports and MIS reports for operational analysis and compliance.



\## Test Automation



\- Designed and executed end-to-end test cases covering critical workflows of the application.

\- Developed automation scripts using \*\*Java\*\* and \*\*Selenium WebDriver\*\* for UI testing.

\- Utilized \*\*TestNG\*\* for test execution, configuration, and reporting.

\- Implemented data-driven testing using \*\*TestNG DataProvider\*\* to handle multiple login scenarios.

\- Covered edge cases to ensure system reliability and robustness.
